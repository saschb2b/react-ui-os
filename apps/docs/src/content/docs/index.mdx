## Three clones, one skeleton

The same apps, window manager, and keyboard shortcuts run under every theme. macOS, Windows, and Ubuntu each mirror their platform down to the window controls, the dock, the menu bar, and the motion. Switch live in the demo above, or open any one on its own.

## The desktop in one tag

```tsx
import { Desktop } from "@react-ui-os/desktop";
import { macosTheme } from "@react-ui-os/theme-macos";

const apps = [{ id: "notes", name: "Notes", content: () => <h1>Hello, desktop.</h1> }];

<Desktop apps={apps} theme={macosTheme} />;
```

Wallpaper, dock, draggable and resizable windows with traffic lights, focus tracking, the minimize-to-dock genie, Cmd-K Spotlight, Cmd-, Settings: all from that tag. Register one app object and it lights up the dock, the menu bar, Spotlight, and the keyboard shortcuts at once. The library composes the system; you describe the apps.

## Prebuilt apps

You don't have to build every app. Prebuilt ones live in a registry; install one with the CLI and the source lands in your codebase, yours to edit.

```bash
npx @react-ui-os/cli add notes
```

[Browse the App Store](/app-store/) or [publish your own](/app-store/publish/).
